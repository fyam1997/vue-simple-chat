# vue-simple-chat

https://fyam1997.github.io/vue-simple-chat/

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Deployment

```sh
npm install
npm run build
gh-pages -d dist -m "$(git rev-parse HEAD)"
```
